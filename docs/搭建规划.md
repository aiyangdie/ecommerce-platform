# 平台搭建规划（执行路线图）

> 目标：对外展示「具备搭建高品质电商平台」的完整工程能力。  
> 仓库：https://github.com/aiyangdie/ecommerce-platform （私有，可按需改为 Public 展示作品集）

## 阶段 0：基础设施（已完成）

- [x] Monorepo：`mall-api` / `mall-admin` / `mall-uni` / `docker`
- [x] MySQL + Redis Docker Compose + 一键安装脚本
- [x] 完整购物流程 + 管理端 + GitHub Pages 演示

## 阶段 1：本地跑通（1 周）

1. 安装 Docker、JDK 17、Node 18、HBuilderX  
2. `docker compose -f docker/docker-compose.yml up -d`  
3. `mall-api` → `mvn spring-boot:run`  
4. `mall-admin` → `npm run dev`  
5. HBuilderX 运行 `mall-uni` 到 H5 / 微信开发者工具  
6. 截图录屏存入 `docs/screenshots/`（可选，用于 README 展示）

1. 仓库 **Settings → Pages → Build and deployment → Source: GitHub Actions**
