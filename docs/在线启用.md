# 3 分钟启用在线演示

## 第一步：打开 GitHub Pages

1. 打开 https://github.com/aiyangdie/ecommerce-platform/settings/pages  
2. **Build and deployment → Source** 选 **GitHub Actions**  
3. 保存

## 第二步：触发部署

- 方式 A：已推送最新代码后，打开 **Actions** → **Deploy Online (GitHub Pages)** → **Run workflow**  
- 方式 B：本地 `git push` 到 `main` 分支自动触发

等待约 2~3 分钟，状态变为绿色 ✓。

## 第三步：访问

| 页面 | URL |
|------|-----|
| 门户 | https://aiyangdie.github.io/ecommerce-platform/ |
| 管理后台 | https://aiyangdie.github.io/ecommerce-platform/admin/ |

## 若仍是 404

| 原因 | 处理 |
|------|------|
| 仓库为**私有** | 改为 **Public**（免费账号私有仓无法对外 Pages） |
| Pages 未选 GitHub Actions | 按第一步设置 |
| Workflow 失败 | Actions 页查看日志，修复后重新 Run |

## 对接真实 API（可选）

仓库 **Settings → Secrets and variables → Actions → Variables**：

- 名称：`VITE_API_BASE_URL`  
- 值：`https://你的API域名`（不要带末尾斜杠）

重新运行 Deploy workflow。管理端将优先请求真实 API，失败时回退演示数据。
